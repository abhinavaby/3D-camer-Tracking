# 🕺 3D Motion Capture Avatar

A real-time 3D avatar that mirrors your body movements using a webcam. Built with **Three.js** for rendering and **MediaPipe** for AI-powered pose estimation.

## ✨ Features
- **Real-Time Tracking:** Translates webcam feed to 3D movement instantly.
- **Stylized Character:** Custom low-poly "chunky" boy model.
- **Dynamic Physics:** Bones and joints scale and rotate based on your pose.
- **Head Tracking:** Calculates head tilt and rotation using ear/nose landmarks.

## 🛠️ Tech Stack
- [Three.js](https://threejs.org) - 3D Graphics Engine
- [MediaPipe Pose](https://github.io) - Body Tracking AI
- [JavaScript (ES6)](https://mozilla.org) - Logic & Math
